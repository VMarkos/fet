Σκοπός

Βασικός σκοπός αυτού του tutorial είναι να αποτελέσει μία μικρή, πλην όμως ευχάριστη, εισαγωγή στη σχεδιάση ιστοσελίδων. Θα αρχίσουμε την εξερεύνησή μας στο Front-end Development μέσα από τη σχεδίαση του δημοφιλούς παιχνιδιού του Ναρκαλιευτή. Στόχοι μας, μεταξύ άλλων, μέσα από αυτή τη διαδικασία είναι και οι εξής:

  • Να εξηγήσουμε τις βασικές διαφορές ανάμεσα στην HTML, τη CSS και τη JavaScript.
  • Να συζητήσουμε τις βασικές δομές καθεμιάς από τις παραπάνω.
  • Να μάθουμε να τις χρησιμοποιούμε συνδυαστικά με στόχο να φτιάξουμε ένα απλό παιχνίδι.

Χρήσιμες πηγές καθ' όλη τη διάρκεια του tutorial θα αποτελέσουν τόσο τα Web Docs της MDN όσο και το φοβερό και τρομερό StackOverflow.

Μάθημα 1ο — Τα Βασικά

Στο πρώτο μάθημα αυτού του tutorial θα ασχοληθούμε κυρίως με βασικά στοιχεία της HTML και της CSS, αλλά και με διάφορα εργαλεία που θα μας βοηθήσουν να αντιγράψουμε το χαρακτηριστικό ρετρό ύφος του Ναρκαλιευτή. Παράλληλα, θα σχολιάσουμε και κάποιες καλές πρακτικές που θα μας βοηθήσουν σε όλη την πορεία του tutorial να γράφουμε κώδικα εν... τάξει.

Μάθημα 2ο — Κάνοντας κλικ

Στο πρώτο μάθημα αυτού του tutorial σχεδιάσαμε ένα ταμπλό ολόιδιο με αυτό του παραδοσιακού παιχνιδιού του Ναρκαλιευτή και κάναμε μία μικρή συζήτηση περί χρωμάτων. Σε αυτό το δεύτερο μάθημα θα ασχοληθούμε περισσότερο με το πώς μπορούμε να εντάξουμε κάποια βασική λειτουργικότητα στο παιχνίδι μας.

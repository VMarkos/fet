Εισαγωγή

Πιθανότατα έχετε ακούσει πολλές φορές να γίνεται λόγος για την HTML, τη CSS, την JavaScript, αλλά τι είναι αυτά τα πράγματα, πέρα από περίπλοκα αρκτικόλεξα και σύνθετες λέξεις που εμπλέκουν νησιά της Ινδονησίας;

Πρώτα λίγο μπλα-μπλα...

Ας πάρουμε τα πράγματα με τη σειρά:

  • Η HTML (HyperText Markup Language) είναι ίσως ο ακρογωνιαίος λίθος του διαδικτύου — κι αυτό δεν είναι υπερβολή. Μέσω της HTML καθορίζουμε τη βασική δομή μίας ιστοσελίδας καθώς και το τι «σημαίνουν» τα όσα αυτή περιέχει (θα γίνει σαφέστερο αυτό στην πορεία).
  • Η CSS (Cascading Style Sheets), όπως αποκαλύπτει δειλά-δειλά και το όνομά της, καθορίζει εν πολλοίς το στυλ το οποίο θα έχει η ιστοσελίδα μας. Τα χρώματα, το πώς θα μοιάζουν αλλά και το πώς θα κινούνται τα πράγματα. Η CSS, με λίγα λόγια, καθορίζει τη μορφή που θα πάρουν τα πράγματα στην οθόνη μας — ή όπου αλλού.
  • Η JavaScript είναι μία εύπεπτη (για τις ηλεκτρονικές συσκευές και, δη, για τους browsers) γλώσσα προγραμματισμού που μας βοηθάει να προσθέσουμε λειτουργικότητα στις ιστοσελίδες μας. Αν και πολλές φορές, στοιχειώδη λειτουργικότητα μπορούμε να προσθέσουμε και μέσω της CSS, ο απλούστερος και πιο αποδοτικός (συνήθως) τρόπος να το κάνουμε είναι μέσω της JavaScript. Να πούμε και κάπου εδώ ότι, παρά το πολύ κοντινό όνομα, η κυρία JavaScript έχει ελάχιστη συγγένεια με την γνωστή και μη εξαιρετέα στους κύκλους του προγραμματισμού, Java.

Ο Ναρκαλιευτής

Τώρα που ξεκαθαρίσαμε κάποια πράγματα, καιρός να αρχίσουμε να ασχολούμαστε με τον Ναρκαλιευτή μας. Σε αυτό το πρώτο μέρος θα εστιάσουμε λίγο στο να αντιγράψουμε το βασικό στυλ του παιχνιδιού χρησιμοποιώντας HTML και CSS. Αρχικά, ας θυμηθούμε πώς έμοιαζε ο Ναρκαλιευτής των Windows XP:

Το θυμάστε;

Πριν ξεκινήσουμε να αντιγράφουμε, ας περιηγηθούμε λίγο στα διάφορα στοιχεία που έχει η παραπάνω εικόνα:

  • Πάνω αριστερά υπάρχει ένας μετρητής που μας ενημερώνει για το πόσες ακόμα νάρκες μένουν να «ανακαλυφθούν».
  • Πάνω δεξιά υπάρχει ένα μικρό και διακριτικό χρονόμετρο που μετράει (σε δευτερόλεπτα), τον χρόνο που παίζουμε.
  • Ανάμεσα στα δύο αυτά χρονόμετρα υπάρχει μία χαρούμενη (προς το παρόν) φατσούλα, η οποία μας ενημερώνει για το status του παιχνιδιού. Για την ακρίβεια, όσο είναι ακόμα χαμογελαστή, δεν έχουμε πατήσει ακόμα κάποια νάρκη, όταν φοράει γυαλιά ηλίου σημαίνει πως έχουμε κερδίσει το παιχνίδι ενώ όταν κάνει την ψόφια σημαίνει ότι έχουμε χάσει.
  • Στο κυρίως μέρος βλέπουμε το κλασσικό ταμπλό του παιχνιδιού, που περιέχει κελιά που δεν έχουν ακόμα αποκαλυφθεί, κελιά που έχουμε σημειώσει ως ναρκοθετημένα καθώς και κελιά που περιέχουν κάποιον χρωματιστό αριθμό μέσα τους. Σε περίπτωση που δεν το θυμάστε, οι αριθμοί υποδηλώνουν το πλήθος των ναρκών που υπάρχουν στα γειτονικά κελιά — σημειώστε το αυτό για επόμενο μάθημα.

Ταμπλό και...μέσα

Εντάξει, οι τίτλοι των ενοτήτων είναι cringe, αλλά έχουν εγκριθεί από τον αρχισυντάκτη μας, οπότε, τι να κάνουμε τώρα! Θα ξεκινήσουμε την περιήγησή μας αντιράφοντας το βασικό ταμπλό του παιχνιδιού, αφήνοντας προς το παρόν τόσο τους μετρητές όσο και τα νουμεράκια στην άκρη.

Κάνοντας την αρχή

Το πρώτο πράγμα που πρέπει να κάνουμε είναι να πάμε κάπου και να φτιάξουμε έναν φάκελο μέσα στον οποίο θα βρίσκεται ό,τι έχει σχέση με το project μας. Δημιουργούμε, λοιπόν, έναν φάκελο, ας τον πούμε minesweeper-tutorial και μέσα σε αυτόν δημιουργούμε ένα αρχείο με το όνομα index.html. Όπως φαντάζεστε, σε αυτό το αρχείο θα γράψουμε τη βασική μας HTML. Αρχικά, προσθέτουμε στο αρχείο τις ακόλουθες γραμμές κώδικα: 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>
            Minesweeper
        </title>
    </head>
    <body>
    </body>
</html>

Οι παραπάνω γραμμές κώδικα δεν έχουν κάποιο ιδιαίτερο περιεχόμενο, πέρα από το ότι αρχικοποιούν, κατά κάποιον τρόπο, το περιεχόμενο της σελίδας μας. Αναλυτικότερα:

  • Η γραμμή <!DOCTYPE html> μας ενημερώνει ότι ο τύπος του εγγράφου είναι HTML. Άλλοι πιθανοί τύποι θα μπορούσσαν να είναι, για παράδειγμα, XML ή SVG.
  • Η γραμμή <html>, που ζευγαρώνει με την τελευταία γραμμή, </html>, οριοθετεί το τμήμα του κώδικά μας που θα περιέχει το κυρίως μέρος της σελίδας μας. Γενικά, οι δομές της μορφής <κάτι> και </κάτι> λέγονται tags, και, όπως θα δούμε και στη συνέχεια, οριοθετούν διάφορες βασικές δομές που χρησιμοποιούμε κατά τη σχεδίαση μίας ιστοσελίδας.
  • Το tag <head> οριοθετεί την κεφαλή της σελίδας μας. Πρακτικά, μέσα σε αυτό το tag γράφουμε διάφορες πληροφορίες που αφορούν όλη τη σελίδα μας. Τέτοιες μπορεί να είναι, για παράδειγμα:
    • Η κωδικοποίηση των χαρακτήρων που θα χρησιμοποιήσουμε, που εδώ καθορίζεται από τη γραμμή <meta charset="UTF-8"> — περισσότερα για την κωδικοποίηση UTF-8 μπορείτε να βρείτε εδώ.
    • Ο τίτλος της σελίδας, ο οποίος είναι αυτό το διακριτικό κειμενάκι που φαίνεται στην καρτέλα των περισσότερων browsers και καθορίζεται με τη βοήθεια του tag <title>.
    • Το «στυλ» της σελίδας, το οποίο συνήθως καθορίζεται από κάποιο αρχείο CSS, όπως θα δούμε και παρακάτω.
    • Κάποιο script, για παράδειγμα, της JavaScript, το οποίο χρειάζεται από κάποιο τμήμα της σελίδας μας — αυτό θα το δούμε σε επόμενο μάθημα.
  • Το tag <body> οριοθετεί το κυρίως μέρος (σώμα) της σελίδας μας, δηλαδή εκεί που θα περιέχονται όλα όσα καθορίζουν τη δομή της.

Όπως φαντάζεστε, αν πάτε τώρα να ανοίξετε το αρχείο αυτό με κάποιον browser, το μόνο που θα δείτε είναι μία κενή σελίδα, καθώς δεν έχουμε τοποθετήσει τίποτα μέσα στο σώμα της σελίδας μας. Πρέπει, λοιπόν, σιγά-σιγά, να αρχίσουμε να βάζουμε περιεχόμενο.

Λίγο στυλ...

Το πρώτο που θα κάνουμε θα είναι να δώσουμε ένα ωραίο χρώμα στο φόντο της σελίδας μας. Αυτό δεν έχει τόσο λειτουργική σημασία, όσο περισσότερο αισθητική. Για να το κάνουμε αυτό, θα χρειαστεί να γράψουμε λίγη CSS. Δημιουργούμε, λοιπόν, έναν φάκελο css μέσα στον φάκελο minesweeper-tutorial και μέσα στον ολοκαίνουργιο φάκελό μας δημιουργούμε ένα αρχείο που το ονομάζουμε style.css. Μέσα σε αυτό το αρχείο γράφουμε τα εξής: 

html, body {
    margin: 0px;
    padding: 0px;
}

html {
    background: #4585ff;
}

Έπειτα, πάμε στην κεφαλή του αρχείου index.html και προσθέτουμε την ακόλουθη γραμμή: 

<!DOCTYPE html>
<html>
    <head>
        ...
        <link rel="stylesheet" type="text/css" href="./css/style.css" media="screen"/>
        ...
    </head>
    ...
</html>

Αν τώρα ανοίξετε (ή ξαναφορτώσετε) το αρχείο index.html στον browser σας, θα δείτε ότι ολόκληρη η καρτέλα έχει πάρει ένα ήπιο μπλε χρώμα. Ας εξηγήσουμε λίγο τι κάναμε παραπάνω:

  • Η γραμμή που προσθέσαμε στον index.html φορτώνει όλο το περιεχόμενο του αρχείου ./css/style.css στη σελίδα μας. Προσέξτε πώς δηλώνουμε ότι το αρχείο βρίσκεται σε έναν υποφάκελο, χρησιμοποιώντας την /.
  • Στο αρχείο style.css καθορίζουμε αρχικά ότι το margin και το padding (όπως λέμε στο χωριό μου) των tags <html> και <body> θα είναι μηδενικό. Το τι είναι αυτά τα δύο πράγματα θα το συζητήσουμε σε πολύ λίγο, αλλά, αν βιάζεστε πια τόσο πολύ, μπορείτε να δοκιμάσετε να σβήσετε αυτές τις δύο γραμμές, και να ξαναφορτώσετε τη σελίδα μας. Τι παρατηρείτε;
  • Παρακάτω καθορίζουμε το χρώμα του φόντου του tag <body> (δηλαδή του σώματος της σελίδας μας), δίνοντας έναν δεκαεξαδικό κωδικό. Γενικά, υπάρχουν πάρα πολλοί τρόποι να κωδικοποιήσουμε τα χρώματα, τους οποίους θα αναλύσουμε στην πορεία των μαθημάτων μας. Για περισσότερα, μπορείτε να δείτε εδώ.

Ώρα, σιγά-σιγά, να αρχίσουμε να περνάμε στην ουσία. Αρχικά, θα δημιουργήσουμε τις «υποδομές» που θα φιλοξενήσουν το ταμπλό του παιχνιδιού μας. Γι' αυτόν τον σκοπό, τροποποιούμε το index.html ως εξής: 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <link rel="stylesheet" type="text/css" href="./css/style.css" media="screen"/>
        <title>
            Minesweeper
        </title>
    </head>
    <body>
        Test
    </body>
</html>

Ακολούθως, προσθέτουμε και στο style.css τα εξής: 

html, body {
    margin: 0px;
    padding: 0px;
}

html {
    height: 100%;
}

body {
    min-height: 100%;
    background: #4585ff;
    display: flex;
    justify-content: center;
    align-items: center;
}

Το ζουμί εδώ βρίσκεται στο τι έχουμε κάνει στην CSS μας. Αρχικά, δηλώσαμε ότι το tag <html> θα πρέπει να πιάνει το 100% της σελίδας μας, γεγονός που θα μας βοηθήσει να «κεντράρουμε» το περιεχόμενό της. Έπειτα, προσθέσαμε κάποιους ακόμα κανόνες για το <body>:

  • Αρχικά, του ζητήσαμε να έχει ύψος τουλάχιστον όσο και η σελίδα μας, μέσα από τον κανόνα min-height: 100%;.
  • Έπειτα, του ζητήσαμε να κάνει...flex, μέσα από τον κανόνα display: flex;. Απλά, όταν λέμε "flex" στην περίπτωσή μας δεν εννοούμε αυτό που ίσως καταλάβατε, αλλά ότι θέλουμε το σώμα να απλωθεί (εκ προοιμίου, οριζοντίως), όσο μπορεί.
  • Με τον κανόνα justify-content: center; ζητάμε από το σώμα της σελίδας μας να τοποθετήσει το περιεχόμενο στο κέντρο (οριζοντίως) ενώ με τον κανόνα align-items: center; τοποθετούμε το περιεχόμενο στο κέντρο (καθέτως).

Ανοίξτε το αρχείο index.html και παρατηρήστε ότι στο κέντρο της μπλε σελίδας μας τώρα υπάρχει ένα «Test». Στη θέση αυτού του θα θέλαμε εμείς να εμφανίσουμε ένα μικρό πλέγμα που να θυμίζει αυτό του ναρκαλιευτή. Αυτό μπορούμε να το πετύχουμε με τις ακόλουθες τροποποιήσεις στα δύο αρχεία μας: 

<body>
    <div class="main-content-container">
        <div class="minesweeper-grid-container">
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
            <div class="minesweeper-cell"></div>
        </div>
    </div>
</body>

Και στο αρχείο style.css: 

.minesweeper-grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr 1fr;
}

.minesweeper-cell {
    width: 50px;
    height: 50px;
    background: red;
    border: 2px solid black;
}

Στο αρχείο index.html δεν προσθέσαμε τίποτα το ιδιαίτερο πέρα από ένα <div> που θα αγκαλιάσει το πλέγμα μας και 16 <div> που θα παίξουν τον ρόλο των κελιών. Στην CSS τώρα, η κλάση .minsweeper-grid-container είναι μία κλάση που αποτελείται από ένα πλέγμα (grid) το οποίο θα έχει, όπως υποδηλώνει και ο κανόνας grid-template-columns: 1fr 1fr 1fr 1fr, θα αποτελείται από τέσσερις στήλες ίσου πλάτους. Το fr δεν είναι παρά μία σχετική μονάδα μέτρησης που χρησιμοποιούμε όταν θέλουμε να δηλώσουμε το πλάτος των στηλών (ή το ύψος των γραμμών) ενός πλέγματος. Αν δοκιμάσετε λίγο να πειραματιστείτε με διάφορους συνδυασμούς θα καταλάβετε ακριβώς πώς δουλεύει — αν δε θέλετε να πειραματιστείτε, πρώτον, ντροπή σας, δεύτερον, μπορείτε να βρείτε περισσότερα εδώ.

Να σημειώσουμε κάπου εδώ πως μία εναλλακτική στον παραπάνω κανόνα, για να μη γράφουμε συνεχώς 1fr είναι το repeat. Έτσι, θα μπορούσαμε να ξαναγράψουμε τον κανόνα μας ως εξής: grid-template-columns: repeat(4, 1fr);. Το συντακτικό είναι αρκετά απλό, καθώς στο πρώτο όρισμα δίνουμε το πλήθος των επαναλήψεων και στο δεύτερο το περιεχόμενο κάθε επανάληψης.

Πάμε τώρα στην κλάση minesweeper-cell. Εκεί έχουμε καθορίσει ρητά το πλάτος (width) και το ύψος (height) κάθε στοιχείου στα 50px ενώ έχουμε δώσει κι ένα ενδεικτικό χρώμα στο φόντο κάθε κελιού. Τέλος, έχουμε καθορίσει το κάδρο του κάθε κελιού (border) να είναι μαύρο (black), να μην έχει κάποιο ιδιαίτερο σχέδιο (solid) και να έχει πλάτος 2px. Αν θέλετε να μάθετε περισσότερα για τις διάφορες μονάδες μέτρησης στην CSS, μπορείτε να ρίξετε μια ματιά εδώ.

Και τα χρώματα;

Ως εδώ, έχουμε φτιάξει ένα κοκκινόμαυρο πλέγμα στη μέση μίας μπλε σελίδας. Όχι και πολύ κοντά στο συνηθισμένο ρετρό στυλ του ναρκαλιευτή μας, θα έλεγε κανείς... Λοιπόν, καιρός να πιάσουμε λίγη σοβαρή δουλειά. Αρχικά, πρέπει να βρούμε ποια χρώματα χρειαζόμαστε για να μοιάζουν τα κελιά μας με αυτά του ναρακαλιευτή. Για να το κάνουμε αυτό βρίσκουμε πρώτα μία εικόνα στο διαδίκτυο (ή όπου αλλού μπορούμε) στην οποία να φαίνονται καθαρά κάποια κελιά από το παιχνίδι μας, όπως για παράδειγμα παρακάτω:

Κάπου την έχουμε ξαναδεί αυτή την εικόνα, ε;

Έπειτα, είτε χρησιμοποιώντας κάποιο πρόγραμμα επεξεργασίας εικόνας είτε κάποιον online color picker, αρχίζουμε να παίζουμε με τα pixels της εικόνας, μέχρι να βρούμε τα χρώματα που μας ενδιαφέρουν. Ενδεικτικά, ένας ενδιαφέρον online color picker είναι ο: imagecolorpicker.com, ενώ σίγουρα τη δουλειά σας μπορείτε να κάνετε και με ένα λογισμικό όπως το GIMP — αναλυτικές οδηγίες μπορείτε να βρείτε εδώ. Με τη βοήθεια, λοιπόν, της τεχνολογίας, τα χρώματα που θα χρειαστούμε για να χρωματίσουμε τα κελιά μας (ακούγεται λάθος, έτσι;) είναι τα εξής:

  • #c6c6c6 για το κελί,
  • #808080 για τη «σκοτεινή» σκιά του κελιού — δεξιά και κάτω,
  • λευκό για τη «φωτεινή» σκιά του κελιού — αριστερά και πάνω.

Αλλάζοντας, λοιπόν, λίγο την κλάση .minesweeper-cell: 

.minesweeper-cell {
    width: 50px;
    height: 50px;
    background: #c6c6c6;
    border-bottom: solid 4px #808080;
    border-right: solid 4px #808080;
    border-top: solid 4px white;
    border-left: solid 4px white;
}

FeT

Τι είναι το FeT;

Έχετε νιώσει ποτέ την ανάγκη να σχεδιάσετε από το μηδέν μία δική σας ιστοσελίδα; Η αλήθεια είναι ότι, για να διαβάζετε αυτό το κείμενο, μπορεί και να την έχετε νιώσει. Ωστόσο, δεν είναι οι βαθύτερες σχεδιαστικές σας παρορμήσεις αυτό που θα συζητήσουμε εδώ. Ή, βασικά, μπορεί και να είναι και αυτό μέσα στα όσα θα μας απασχολήσουν. Τέλος πάντων, όπως λέγαμε και παραπάνω...

«Καααααατ! Πάλι από την αρχή! "Τι είναι το FeT", Σκηνή 1η, Λήψη 46. Πάμε!»

Για να βρίσκεστε σε αυτόν τον ιστότοπο, είτε σας ενδιαφέρει το Front-end Development, είτε σας έστειλε κάποιος έναν σύνδεσμο μαζί με ένα μήνυμα του τύπου: «Κοίτα, μία σελίδα που θυμίζει καρπούζι!». Όπως κι αν έχει, εδώ δε θα βρείτε καρπούζια, τουλάχιστον όχι τόσο συχνά. Αυτό που θα βρείτε είναι διάφορα μικρά tutorials που σχετίζονται με τη σχεδίαση και ανάπτυξη ιστοσελίδων. Καθένα από αυτά τα tutorials είναι οργανωμένο σε επιμέρους μαθήματα κι έχει ως στόχο του να παρουσιάσει και να φωτίσει κάποια συγκεκριμένα, κάθε φορά, στοιχεία του Front-end Development.

Θα μπορούσαμε να μακρυγορούμε για ώρες, αλλά, επειδή τα πολλά λόγια είναι φτώχεια και ο σκηνοθέτης γκρινιάζει και είναι πολύ ευέξαπτος, αν σκρολλάρετε λίγο πιο κάτω, θα βρείτε τα tutorials που αναφέραμε — η λίστα ανανεώνεται συνεχώς, οπότε stay tuned!
